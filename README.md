# koydo-pte-academic

PTE Academic exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM009 |
| Name | PTE Academic |
| Country | Global |
| Category | Language proficiency |
| Tier | Tier 1 |
| Wave | Wave 2 |
